Sobre Nosotros

Nuestra Misión

Empoderar a las personas para que alcancen sus objetivos de salud y bienestar a través de planes de nutrición personalizados que sean accesibles, efectivos y disfrutables.

Nuestra Visión

Revolucionar la forma en que las personas abordan la nutrición combinando tecnología de vanguardia con ciencia nutricional.

Nuestros Valores

Personalización

Creemos que no hay dos personas iguales, y sus planes de nutrición tampoco deberían serlo.

Adaptabilidad

Aprendemos y nos ajustamos continuamente para ofrecerte las soluciones más efectivas.

Simplicidad

Hacemos que la nutrición sea sencilla y accesible, eliminando la complejidad.

Información Técnica

PlaniMe Web Client, desarrollada por el Ing. Diego Magaña Álvarez, es una interfaz frontend que consume los servicios de PlaniMe RESTful API (desarrollada por el mismo) mediante solicitudes HTTP asíncronas, siguiendo el principio de separación de preocupaciones.

Arquitectura Cliente-Servidor

Backend

  • Gestión de lógica de negocio con controladores modulares
  • Autenticación/autorización mediante JWT (RFC 7519) y middleware de seguridad
  • Persistencia de datos relacional con validación estricta

Frontend

  • Responsable de la representación visual (UI)
  • Gestión de la interacción del usuario (UX)
  • Comunicación mediante endpoints RESTful estructurados

Estándares y Características

  • Cumple con estándares OWASP para seguridad
  • Arquitectura REST para interoperabilidad
  • Diseñado para escalabilidad y mantenibilidad
